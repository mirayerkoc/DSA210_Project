# KNOWN COMPATIBILITY ISSUES & SOLUTIONS
# =============================================================================
#
# 1. LightGBM on Windows:
#    Problem: May fail due to missing Visual C++ Redistributable
#    Solution: Download and install from:
#              https://visualstudio.microsoft.com/downloads/
#
# 2. XGBoost on older systems:
#    Problem: Compilation errors on Python 3.7 or older
#    Solution: Upgrade to Python 3.8+ or use older XGBoost:
#              pip install xgboost==1.7.6
#
# 3. Apple Silicon (M1/M2/M3):
#    Problem: pip wheels may not be optimized
#    Solution: Use conda for native ARM64 builds:
#              conda install -c conda-forge xgboost lightgbm scikit-learn
#
# 4. NumPy/SciPy installation errors:
#    Problem: Missing BLAS/LAPACK libraries
#    Solution: Install via conda:
#              conda install numpy scipy
#
# =============================================================================
# PERFORMANCE OPTIMIZATION (OPTIONAL)
# =============================================================================
#
# For faster linear algebra operations:
#   - Intel MKL (Intel CPUs):     conda install mkl mkl-service
#   - OpenBLAS (All platforms):   conda install openblas
#
# For parallel processing:
#   - Use n_jobs=-1 in scikit-learn (uses all CPU cores)
#   - XGBoost automatically uses multithreading
#   - LightGBM has built-in parallelization
#
# Memory optimization:
#   - Use pandas dtypes efficiently (int32 vs int64, category dtype)
#   - LightGBM uses less memory than XGBoost
#
